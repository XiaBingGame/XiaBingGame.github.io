第7章 模式匹配与正则表达式

  • re 模块
  • \d 数字, {数字} 重复次数
  • re.compile() 加上正则表达式得到一个Regex Pattern 对象. 该对象有 search() 方法, 如果有匹配, 则返回 Match 对象. 该对象有 group() 方法, 返回实际的匹配文本.
  • 添加括号可以分组正则表达式
  • '|' 匹配多个表达式
  • 正则模式的findall() 方法发现所有匹配, 返回一个列表
  • ? 字符表示可选, 匹配 0 次或 1 次
  • * 字符表示匹配0次或多次
  • + 字符表示匹配1次或多次
  • re.DOTALL 作为 re.compile() 的第二个参数, 让 '.' 匹配所有字符, 包括换行字符.
  • re.IGNORECASE 不区分大小写
  • sub() 替换字符串
  • re.VERBOSE 忽略正则表达式字符串中的空白符和注释.

第8章 读写文件

  • os.path.join() 返回一个文件路径的字符串, 包含正确的路径分隔符 
    >>> import os
>>> os.path.join('usr', 'bin', 'spam')
'usr\\bin\\spam'
  • os.getcwd() --- 取得当前的工作路径
  • os.chdir() --- 改变当前的工作路径
  • os.makedirs() --- 创建新文件夹, 会自动创建所有必要的中间文件夹
  • os.path 模块
  • os.path.abspath(path) 返回绝对路径
  • os.path.isbas(path) 是否绝对路径
  • os.path.relpath(path, start) 返回相对路径
  • os.path.dirname(path) 返回目录
  • os.path.basename(path) 返回最后一个斜杠之后的所有内容.
  • os.path.split(path) 获取目录和basename
  • os.path.getsize(path) 得到 path 参数中文件的字节数
  • os.listdir(path) 得到目录中文件的列表.
  • os.path.exists(path) 文件或文件夹是否存在
  • os.path.isfile(path) 是否文件
  • os.path.isdir(path) 是否目录
  • open() 打开文件
  • File 对象 read(), readlines() 方法. write() 方法.
  • shelve 模块保存变量 
    import shelve
shelfFile = shelve.open('mydata')
cats = ['Zophie', ' Pooka', 'Simon']
shelfFile['cats'] = cats
shelfFile.close()
  • pprint.pformat() 保存变量 
    >>> import pprint
>>> cats = [{'name: 'Zophie', 'desc': 'chubby'}, {'name: 'Pooks', 'desc': 'fluffy'}]
>>> pprint.pformat(cats)
>>> fileobj = open('myCats.py', 'w')
>>> fileobj.write('cats = ' + pprint.pformat(cats) + '\n')
>>> fileobj.close()
