第三章 函数的增长

  • 渐进符号
    • 圆H记号: OH(g(n)) = {f(n): 存在正整数c1, c2和n0, 使对所有的n>=n0, 有 0<= c1*g(n) <= f(n) <= c2*g(n)}
    • 大O记号: O(g(n)) = {f(n): 存在正整数c和n0, 使对所有的n>=n0, 有 0<= f(n) <= c*g(n)}, 表示最坏情况下的运行时间
    • 欧姆记号: Ω(g(n)) = {f(n): 存在正整数c和n0, 使对所有的n>=n0, 有 0<= c*g(n)} <= f(n), 表示最坏情况下的运行时间
  • 定义3.1: 任意两个函数 f(n), g(n), 如f(n)=OH(g(n)), 则f(n)和g(n)存在大O和欧姆关系
