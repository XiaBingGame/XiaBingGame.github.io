文件和目录 描述
_config.yml 存储配置数据
_draft 不发表的 post, 这些文件的格式没有日期: title.MARKUP
_includes 通过你的 layouts 和 posts 混合匹配该部分, liquid tag {% include file.ext %} 可用于在 _includes/file.ext 内包含该部分
_layout 包装 post 的模板. 在 YAML Front Matter 中选择 layout. liquid tag 用于插入内容
