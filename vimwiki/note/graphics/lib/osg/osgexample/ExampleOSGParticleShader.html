  • API
    • osgParticle::ParticleSystem
      • 派生自 osg::Drawable
      • setVisibilityDistance 设置可视距离
    • osgParticle::Particle
      • Particle 由 Emitter 创建, 而后放入粒子系统中
      • 可永久生存, 只需 lifeTime < 0
      • 属性的当前值可以用一个范围值表示, 当 t = 0 时, 当前值等于 min, 当 t = liftTime, 当前值等于 max
        • 可以自定义 interpolator 对象达到某种类型的过渡效果
        • 当 lifeTime < 0 时, 不会有任何插值, 只会使用 min 值
      • setShape 形状
      • setSortMode 设置排序模式
